European Languages

Snap Backup is an open source personal backup utility that is available in various European languages.

Supported European Languages:
Albanian (Shqip)
Dutch (Nederlands)
English
French (Français)
German (Deutsch)
Italian (Italiano)
Portuguese (Português)
Romanian (Română)
Spanish (Español)
Slovene (Slovenščina)

If you want to see Snap Backup offered in a new language, consider volunteering to translate Snap Backup into your language.
